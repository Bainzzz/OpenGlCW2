*to run game without VS. extract zipped deliverable file, then extract the 'runnable.zip' file, Open 'OpenGLCW2.exe', the game should run.*

*to open the VS,  extract zipped deliverable file, then extract the 'OpenGLCW2.zip' file, the 'OpenGLCW2.sln' should be present.*

*Controls*

*WASD - Move*

*Left mouse click - Shoot*

*ESC - Close game*
